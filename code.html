Coding Toolkit

Small tools for everyday coding.

I kept finding myself bouncing between different tools for simple formatting tasks. AI assistants handle most of these tasks now, but I still use this as a reliable place for the quick converters and helpers I reach for more often than expected.

Formatter

List converter

Turn line-separated values into a comma-separated list, or send a comma-separated list back into one value per line.

SQL helper

SQL list builder

Paste IDs, names, or codes and format them for a SQL lookup without opening a separate formatter site.

JSON helper

JSON builder and formatter

Turn a pasted list into a JSON array, or pretty-print and minify JSON payloads without leaving the page.

Encoder

Base64 encoder and decoder

Encode plain text into Base64 or decode Base64 back into readable text.

API helper

API request builder

Build a URL with query parameters, clean up request headers, or generate a quick cURL command.
